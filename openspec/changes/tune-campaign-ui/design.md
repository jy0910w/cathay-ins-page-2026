# Design: Campaign UI Styling

## 背景與目標

根據設計稿「國泰產險_舞夠有禮_打包_0120.jpg」調整活動頁面視覺樣式，確保與設計稿一致。

## 關鍵決策

### 1. 色彩系統

**主色：**
- **黃色：** `#FFC83B` - 所有按鈕背景、標題強調文字、徽章
- **黑色：** `#3F3F3F` - 主要文字色、按鈕文字
- **綠色：** `#E6F9EF` - Steps 區背景（含點點紋理）
- **白色：** `#ffffff` - Hero 區背景、卡片背景

**設計原則：**
- 全站統一使用單一黃色色號 `#FFC83B`
- 避免使用顏色類別（如 `.btn_orange`），直接在元素選擇器定義
- 黑色文字統一使用 `#3F3F3F` 確保對比度與可讀性

### 2. 區塊結構與背景

**Hero Section (白色) → 波浪裝飾 (綠色) → Steps Section (綠色)**

```
┌─────────────────────┐
│  Hero (白色)        │
└─────────────────────┘
      ╱‾‾‾‾‾╲ ← 綠色梯形波浪
┌─────────────────────┐
│  Steps (綠色+紋理)  │
└─────────────────────┘
```

**波浪裝飾實作：**
使用 CSS `clip-path` 製作梯形波浪：
```css
#steps::before {
  clip-path: polygon(0 100%, 40% 100%, 50% 0, 60% 100%, 100% 100%);
  /* 左右各 40% 平行，中間 20% 三角凸起 */
}
```

**優點：**
- 純 CSS 實作，無需額外圖片
- 響應式自適應（手機 60px，桌機 100px）

### 3. 按鈕佈局

**Flexbox 平均分配寬度：**
```css
.hero_cta {
  display: flex;
  max-width: 500px;
  gap: 1rem;
}

.btn {
  flex: 1; /* 平均分配空間 */
  min-width: 150px; /* 最小寬度保護 */
}
```

**箭頭符號：**
使用 `::after` 偽元素，避免 HTML 實體：
```css
.btn::after {
  content: '>';
  position: absolute;
  right: 1rem; /* Hero/Prizes 區 */
}

.step_btn::after {
  right: 0.75rem; /* Steps 區 */
}
```

**優點：**
- 視覺整齊（兩按鈕等寬）
- CSS 獨立控制箭頭樣式
- 手機版自動垂直排列

## 風險與考量

### 潛在問題：
1. **波浪裝飾的瀏覽器相容性**
   - `clip-path` 在 IE 不支援
   - 緩解：現代瀏覽器皆支援，IE 會降級顯示直線

2. **按鈕文字過長**
   - 超過 `min-width` 會自動撐寬
   - 緩解：已設定 `max-width: 500px` 限制容器總寬
